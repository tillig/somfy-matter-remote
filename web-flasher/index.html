Flash the Somfy Awning Matter Remote

This installs the firmware onto an Elegoo ESP32 DevKit V1 directly from your browser. No build tools or command line are needed.

Use a desktop version of Chrome or Edge. This page relies on Web Serial, which Firefox, Safari, and mobile browsers do not support.
  1. Connect the ESP32 to this computer with a USB data cable.
  2. Click Connect and Install below.
  3. Choose the ESP32's serial port when the browser asks.
  4. Wait for the install to finish, then follow the on-device Wi-Fi setup.

Your browser does not support Web Serial. Use desktop Chrome or Edge. Serial access was blocked. Allow it and reload the page.

After flashing, the device hosts an Awning-Setup-XXXX Wi-Fi network for first-time configuration. See the commissioning guide for the full setup walkthrough, and the project README for an overview.
